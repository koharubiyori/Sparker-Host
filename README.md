## Sparker

The Windows client of [Sparker app](https://github.com/koharubiyori/Sparker).

Sparker is an Android project that allows you to remotely execute power actions such as boot, shutdown, and more on your computer.

The file size of the client program may be slightly larger than its current features because it includes Electron (a UI framework). However, I also plan to add more UI features in the future, so this is necessary.
